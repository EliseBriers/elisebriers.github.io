Introduction

Trash Raccoons is a multiplayer couch coop game created during the Game Projects course at Digital Arts and Entertainment. With a group of 3 people (2 artists and 1 programmer) we where tasked to create a game together.

In Trash Raccoons, players play as raccoons that try to collect as much trash as possible and deliver this trash back to the garbage truck. Players can work in teams or alone, and can steal trash from other players.

Learned Skills

The development of this project has thought me the following skills:

  • Communicating and working in a team
  • Programming intuitive tools for non-programmers
  • Agile development
  • Working with the Unity game engine
  • C#
